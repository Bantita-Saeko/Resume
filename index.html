SAP Build Apps Mini Project. Low-Code

This project outlines a Student Data Management Application prototype, ideally developed using a low-code platform like SAP Build Apps, to efficiently handle core administrative functions. The system's primary goal is to provide a clean, mobile-friendly interface for managing student records. Key functionality includes a Student List View which displays essential details like ID, name, contact, and academic status, alongside integrated controls for searching, editing, and deleting individual records. A separate Add Student Data screen offers a dedicated form for capturing new student details, including mandatory fields for ID and name, along with dropdowns for academic year and current status, ensuring comprehensive data collection. This prototype emphasizes a user-friendly design to streamline the entire student registry process.
